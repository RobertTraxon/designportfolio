Problem

I was approached by the partnerships team and asked to provide a variety of early years digital activity templates. These would be used to help promote the cleaner seas project in association with ASDA and Peppa Pig.

Research

The research for this project was to distinguish how early years learners were using other Twinkl products and identify how these could be translated to suitable digital adaptations.

Here were some of the key questions I considered when gathering data for this project:

  1. Which Twinkl Go products did the children enjoy?
  2. How could the current Twinkl Go products be more suitable for a younger market.
  3. Which paper-based Twinkl products are most used by EYFS?

Many users enjoyed the Twinkl hotspot activities but found difficulty reading the text.
Puzzle games were a firm favourite amongst all the children but our existing templates were slightly too difficult.
Reading ability is minimal, so reducing reliance on text and simplifying the user experience is important.
3/10 out of the top downloaded puzzle activities were a variation of the sequencing strips game.
Another popular resource type was a 'What can you see?' PowerPoint game.
User Needs
Minimalism
Keeping the UI and UX as minimalistic as possible for the younger audience.
Familiarity
Despite it being a partnership resource it was important to keep it consistent with the existing Twinkl game UI.
Business Needs
Speed
This was a project with a tight deadline, so it needed to be achievable within a short period of time.
Adaptability
To make it cost-effective, being able to adapt the template into other variations in the future was critical.
Customisation
To make it fit with the Peppa Pig brand, additional features such as customisable sound effects would need to be implemented.
Solution

Create 2 interactive templates reflecting current EYFS interests and abilities as well as adapt the existing templates to better suit younger learners.

Puzzle Strips

The first template would be inspired by the sequencing strips activity available on the website as a paper-based game. It would also act as a simplified version of the picture puzzle game that was popular among the test groups.

Binocular Activity

The second game would be an updated version of the PowerPoint "What can you see?" game that would optimise the resource production process and provide essential improvements for the end users too.

One of the adaptations to the existing templates was to remove levelled gameplay and revert to the most simple form. Although levels can be beneficial to older children, the research showed they were less valuable to children under the age of 5.

Before
After

As the puzzle strips template was intended to be a simplified version of my previous picture slider puzzle game, it was important to remove as much clutter as possible. Removing the menu system and using a linear progression through the different images would remove the need for additional triggers on the game screen and simplify the user-flow. The text prompt was also removed, and the grid system was changed to a uni-directional format to make it easier for younger learners to complete the puzzle.

Before
After

The binocular template was designed to replace the PowerPoint version with a couple of key improvements. The binocular cutout would now be tracked to the mouse or touch events instead of an automated sweep. A reveal button would allow the user to move at their own pace, and customisable binocular size would improve usability for a wider age variety.

Increase Productivity

Another aspect I wanted to improve was how long each resource would take to make. Each template would be linked to an online form system so Twinkl staff could simply replace the images and text and select the style and colour from pre-determined dropdowns. This reduced resource creation time by around 90% for the binocular game compared with its PowerPoint counterpart.
